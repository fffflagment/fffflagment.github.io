flagment

fragments, not biography.

flagment は、実名や経歴を並べるためではなく、名前のない誰かの視点だけを置いておくための小さな場所です。
ここにあるのはプロフィールではなく、断片のほう。文脈はほとんど提供されません。

About

これは「自己紹介」ではなく、「輪郭の方針」です。

誰かを特定するための情報は、ここにはほとんどありません。

このページは、一人の人間の全体像を説明することを目的としていません。むしろ、 説明しきらないこと を優先しています。

年齢、居場所、肩書きは、読み手にとっては副次的なものだという前提で作られています。 代わりに、小さな違和感や、温度だけが伝わればそれで十分です。

ここに並ぶテキストや断片は、「本当の全部」ではなく、そのときたまたま切り出された一部にすぎません。 その不完全さのまま、静かに置いておくことを目的にしています。

Fragments

いくつかの断片。意味づけは誰のものでもない。

実際に存在するものと、まだ存在していないものが混ざっています。
visual / glitch

glitch study 01

乱れた画面やノイズを、あえて修復しないまま眺めるためのメモ。 完成度よりも「崩れかけ」の状態のほうが落ち着くときの記録。

text / echo

echo text

誰かとの会話から、文脈を外した一文だけを拾い集めたテキスト。 誰の言葉だったかは、ほとんどの場合もう思い出せません。

ui / night

night interface

黒とわずかな光だけで構成されたインターフェースのスケッチ。 実装される予定のない UI のためのデザイン。

log / mood

static feelings

説明を足さずに、「いまの気分」を一行だけで残していく試み。 蓄積することだけを目的にした、低解像度の記録。

questions

unasked questions

口に出されなかったまま流れていった問いだけを集めたリスト。 回答欄は、ほとんど空白のままです。
